# Better Auth Skill

## Description
Better Auth authentication skill with complete UI components for Next.js applications. Provides full-stack authentication solution with JWT token support, user management, and secure API integration.

## Capabilities
- Complete Better Auth setup with UI components
- User registration, login, and password reset flows
- JWT token configuration for API authentication
- User session management
- OAuth provider integration (Google, GitHub, etc.)
- Protected route handling
- API route authentication with user verification
- User profile management
- Email verification workflows

## Usage Examples
- "Set up Better Auth with login/signin components"
- "Configure JWT tokens for API authentication"
- "Create protected routes using Better Auth"
- "Implement user profile management UI"
- "Add OAuth providers to Better Auth setup"

## UI Components
- Login form with validation
- Sign up form with password requirements
- Password reset flow
- User profile management
- Account settings interface
- Social login buttons
- Loading and error states
- Email verification UI

## Integration Points
- Next.js App Router with middleware protection
- FastAPI backend token verification
- Database user models
- Email service for verification/password reset
- Frontend state management

## Technology Stack
- Better Auth for authentication
- Next.js 15 with App Router
- React Server Components and Client Components
- TypeScript
- Tailwind CSS for styling
- Neon PostgreSQL database
- FastAPI backend for token validation
